A simple and powerful numerical library for linear algebra, optimization, and scientific computing.

Get Started

Features

Linear Algebra

Matrix operations, decompositions, and solvers.

Optimization

Gradient descent, convex solvers, and more.

Statistics

Distributions, sampling, and inference tools.

Installation

Install via npm:

npm install numlib

Or include via CDN:

<script src="https://cdn.example.com/numlib.js"></script>

Example Usage

import { matrixMultiply } from 'numlib';

const A = [[1,2],[3,4]];
const B = [[5,6],[7,8]];

console.log(matrixMultiply(A, B));
